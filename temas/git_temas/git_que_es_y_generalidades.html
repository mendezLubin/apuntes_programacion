➤ git que es y generalidades

-Es un software, sin interfaz grafica (osea que no lo vez)

-Git es una tarjeta de memoria, como en los videojuegos quieres guardar tu proceso.

-Debes en cuando gracias a git vas a ingresar comandos para guardar tu proceso.

-Se ocupa instalar en windows. En Mac o linux generalmente ya viene instalado.

-Se instala solo busca git y lo bajas.

-No tiene versiones lts, la que este es la que instalas.

-Al instalar git se instala gitbash que es su linea de comandos

-comprobar que lo tienes:

   git --version

-Se controla con la linea de comandos gitbash

-Se guardan cambios y crear archivos tambien es un cambio
